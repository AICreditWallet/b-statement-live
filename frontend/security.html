Security

Plain English: how we protect your data and your account.

Secure storage

  • We store data in a managed database with restricted access.
  • Secrets (API keys) are stored on the server (not in the frontend).

Secure access

  • You must login to analyse invoices and view saved results.
  • Users can only access their own data.

File handling

  • We accept common invoice formats (PDF, PNG, JPG/JPEG, HEIC/HEIF).
  • We limit file size for reliability.

Best practice for customers

  • Use a strong password and don’t share login details.
  • Log out on shared devices.
